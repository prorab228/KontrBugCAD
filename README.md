# КонтрБагCAD - 3D САПР редактор онлайн

Онлайн 3D редактор для работы с чертежами и примитивами.

## Возможности
- Создание 3D моделей из примитивов
- Черчение скетчей
- Вытягивание скетчей в 3D
- Булевы операции (объединение, вычитание, пересечение)
- Экспорт в STL/JSON

## Используемые технологии
- Three.js
- JavaScript
- HTML5 Canvas
